# Revalora IA - Revalorizando el Sector Agrícola

**Revalora IA** es una plataforma integral diseñada para transformar el panorama de la agroindustria mediante la economía circular. Nuestra misión es dar una "segunda vida" a los residuos y subproductos agrícolas, convirtiéndolos en recursos valiosos y nuevas oportunidades de negocio.

## 🌟 El Proyecto
En el sector agrícola, gran parte de la producción se pierde o se descarta. Revalora IA utiliza inteligencia artificial para identificar el potencial oculto en estos materiales, facilitando su comercialización y proporcionando el conocimiento técnico necesario para su transformación.

## 🚀 Funcionalidades Principales

### 1. Marketplace Agrícola
Un mercado especializado donde las empresas generadoras de residuos pueden listar sus artículos y subproductos.
*   **Venta de Residuos:** Publicación ágil de mermas y productos no conformes.
*   **Búsqueda Inteligente:** Filtros por tipo de cultivo, ubicación y volumen.
*   **Conexión Directa:** Facilita el trato entre el productor inicial y el comprador interesado en la revalorización.

### 2. EcoInnovador (Asistente IA)
Nuestro asistente inteligente acompaña al usuario en el proceso creativo de reutilización.
*   **Generación de Ideas:** Sugiere usos innovadores para productos específicos (ej. "tengo 5 toneladas de cáscara de naranja, ¿qué puedo hacer?").
*   **Recomendaciones Personalizadas:** Se adapta al tipo de material y volumen disponible.
*   **Interactividad:** Resolución de dudas sobre procesos de transformación de manera inmediata.

### 3. Motor de Recomendaciones y Análisis Técnico
La joya de la corona de Revalora IA. Un sistema avanzado que profundiza en la viabilidad de cada idea de revalorización.
*   **Conceptos de Fabricación:** Guía detallada sobre los pasos necesarios para la producción.
*   **Análisis de Complejidad:** Evaluación del nivel de dificultad técnica y requerimientos de maquinaria.
*   **Estudio de Mercado:** Información sobre la demanda actual, competidores potenciales y precios estimados para el producto final.
*   **Gestión de Ideas:** Posibilidad de guardar y hacer seguimiento a las mejores propuestas analizadas.

## 🌍 Impacto
*   **Sostenibilidad:** Reducción drástica de desperdicios en el campo.
*   **Rentabilidad:** Generación de ingresos adicionales para agricultores y empresas procesadoras.
*   **Innovación:** Fomento de la industria biotecnológica y de materiales sostenibles.
*   **Eficiencia:** Optimización de la cadena de suministro agroindustrial.

## 🛠️ Stack Tecnológico

### Backend
*   **Lenguaje:** .NET 8 / C#
*   **Arquitectura:** Clean Architecture
*   **Base de Datos:** PostgreSQL con Entity Framework Core
*   **Documentación:** Swagger / OpenAPI

### Frontend
*   **Framework:** Angular 21
*   **Estilos:** Vanilla CSS / Tailwind CSS (según configuración)
*   **Estado:** NgRx / Facade Pattern

## 🏁 Inicio Rápido

### Requisitos
*   .NET SDK 8.0+
*   Node.js 20+
*   PostgreSQL

### Ejecución Backend
```powershell
dotnet restore ReciclaYa.sln
dotnet run --project ReciclaYa.Api
```
*API disponible en: `http://localhost:5021/swagger`*

### Ejecución Frontend
```bash
cd frontend
npm install
ng serve
```
*Accede a: `http://localhost:4200`*
